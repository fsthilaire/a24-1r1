USBDeview

USBDeview est un logiciel qui permet d'obtenir de l'information pour tous les périphériques USB actuellement connectés ou qui ont déjà été connectés à l'ordinateur.

Ce logiciel permet de connaitre la version du USB du périphérique.

Pour plus d'information, veuillez consulter le site de NirSoft.

1. Téléchargement​

Téléchargez la dernière version sur le site de Nirsoft

2. Installation​

Il s'agit d'un logiciel portable.

Veuillez extraire le fichier zip dans un dossier.

Je recommande le dossier c:\outils\USBDeview.

3. Windows PE - WinPEx64_A24.iso​

Le logiciel est intégré dans la version de Windows PE qui est utilisé dans ce cours.

Il suffit d'inscrire la commande ci-dessous.

usbdeview
remarque

L'utilitaire se retrouve x:\Programs\usbdeview\USBDeview.exe.

Ce dossier est enregistré dans la variable %path%. C'est pour cette raison que l'exécutable est accessible dans n'importe quel dossier de travail.

L'extension .exe est dans la variable %pathext%. C'est pour cette raison qu'il n'est pas nécessaire d'inscrire usbdeview.exe au complet.

4. Triez les périphériques​

Démarrez le programme USBDeview.exe.

Pour faciliter la lecture, triez les périphériques pour avoir ceux connectés en premier.

Cliquez sur la colonne Connected 2 fois.

Les périphériques connectés sont en premier sur fond vert.

5. Identifier la version en cours d'utilisation​

Pour s'assurer d'avoir la bonne version du périphérique, il faut que celui-ci soit branché dans un port USB de version égale ou supérieure au périphérique. Utilisez toujours un port USB 3.0 pour les périphériques.

L'information de la version se retrouve dans la colonne USB Version.

L'exemple ci-dessus indique que la clé USB DataTraveler 3.0 fonctionne actuellement en 3.10.

attention

Dans l'exemple ci-dessous, la même clé USB est branchée dans un port USB 2.0. La version est maintenant 2.10, car la clé est limitée par la version du port.

La clé USB fonctionnera à une vitesse réduite.
