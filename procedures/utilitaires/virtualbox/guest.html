Installer l'outil des Additions invité

L'outil des Additions invité ou Guest addition tool. est un logiciel qui permet de faire le lien entre la machine virtuelle et la machine hôte.

Celui-ci configure les pilotes des périphériques virtuelles pour avoir une meilleure intégration avec le matériel physique.

Windows​

L'exemple ci-dessous est pour Windows 11. La séquence peut varier un peu selon la version de Windows.

Les étapes 3 à 5 ne sont peut-être pas nécessaires. Le programme d'installation peut démarrer automatiquement lorsque le CD est détecté par le système d'exploitation.

  1. Démarrez la machine virtuelle.

  2. Sélectionnez le menu Périphériques -> Insérer l'image CD des Additions invité....

    Cette action assigne un ISO au lecteur CD de la machine virtuelle.

  3. Dans la machine virtuelle, ouvrez l'Explorateur de fichiers et allez dans Ce PC.

  4. Ouvrez le lecteur de CD.

  5. Exécutez le programme VBoxWindowsAdditions.exe.

  6. Dans la fenêtre d'installation, appuyez sur le bouton Suivant >.

  7. Conservez le dossier d'installation par défaut. Appuyez sur le bouton Suivant >.

  8. Appuyez sur le bouton Installer.

  9. Sélectionnez l'option Redémarrer maintenant et appuyez sur le bouton Fermer.

  10. Après le redémarrage, l'icône de l'outil sera dans la zone de notification.

2. Ubuntu​

L'exemple ci-dessous est pour Ubuntu 22.04. La séquence peut varier un peu selon la version de Ubuntu.

  1. Démarrez la machine virtuelle.

  2. Sélectionnez le menu Périphériques -> Insérer l'image CD des Additions invité....

    Cette action assigne un ISO au lecteur CD de la machine virtuelle.

  3. Cliquez sur l'icône de DVD dans le menu de droite.

  4. Effectuez un clic-droit dans une section vide du contenu des fichiers et sélectionnez l'item Ouvrir dans un terminal.

  5. Dans le terminal, exécutez la commande sudo ./VBoxLinuxAdditions.run

    Important

    Ubuntu est sensible à la casse. Il faut écrire la commande parfaitement avec les majuscules et minuscules.

  6. Entrez le mot de passe.

    Avertissement

    Sous Ubuntu, lorsqu'il faut inscrire le mot de passe dans le terminal, il n'y a aucun indicateur (*****) pour spécifier qu'un caractère est inscrit.

  7. Lorsque l'installation est terminée, redémarrez la machine virtuelle.
