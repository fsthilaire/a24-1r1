Créer un instantané

Note

La machine virtuelle peut être en cours d'exécution lorsqu'on crée un instantané.

  1. Sélectionnez la machine virtuelle.

  2. Appuyez sur le bouton des options de la machine virtuelle.

  3. Appuyez sur l'option Instantanés.

  4. Appuyez sur le bouton Prendre.

  5. Dans la zone Nom de l'instantané, inscrivez le nom de l'instantané.

  6. Dans la zone Description de l'instantané, il est possible d'inscrire un texte détaillé. Cette section est optionnelle.

  7. Appuyez sur le bouton OK. La création peut prendre quelques minutes.

  8. L'instantané se trouve dans la liste.
