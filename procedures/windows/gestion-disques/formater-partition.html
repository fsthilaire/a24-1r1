Formater une partition

Le formatage d'une partition permet de réinitialiser la partition. Tous les fichiers seront effacés.

La taille d'origine est conservée, mais il est possible de modifier le nom et son système de fichiers.

  1. Effectuez un clic-droit sur la partition à formater.

  2. Sélectionnez l'item Formater....

  3. Inscrivez le nouveau nom dans la zone Nom du volume (orange).

  4. Inscrivez le nouveau système de fichiers dans la zone Système de fichiers (vert).

  5. Appuyez sur le bouton OK.

  6. Appuyez sur le bouton Oui pour confirmer le formatage.

  7. Dans l'utilitaire Gestion du disque, la partition a maintenant son nouveau nom et son nouveau système de fichiers.
