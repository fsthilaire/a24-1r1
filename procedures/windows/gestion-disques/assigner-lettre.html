Assigner une lettre à une partition

Il est possible d'assigner ou de modifier une lettre à une partition existante.

Assigner une lettre​

  1. Effectuez un clic-droit sur la partition sans lettre.

  2. Sélectionnez l'item Modifier la lettre de lecteur et les chemins d'accès....

  3. Appuyez sur le bouton Ajouter....

  4. Sélectionnez la lettre du lecteur dans la liste déroulante Attribuer la lettre de lecteur suivante :.. Appuyez sur le bouton OK

  5. Dans l'utilitaire Gestion du disque, la lettre est affichée pour la partition.

Modifier la lettre existante​

  1. Effectuez un clic-droit sur la partition à modifier sa lettre.

  2. Sélectionnez l'item Modifier la lettre de lecteur et les chemins d'accès....

  3. Appuyez sur le bouton Modifier....

  4. Sélectionnez la nouvelle lettre du lecteur dans la liste déroulante Attribuer la lettre de lecteur suivante :.. Appuyez sur le bouton OK

  5. Appuyez sur le bouton Oui dans la fenêtre du message d'avertissement.

  6. Dans l'utilitaire Gestion du disque, la nouvelle lettre est affichée pour la partition.

Supprimer une lettre​

  1. Effectuez un clic-droit sur la partition à supprimer sa lettre.

  2. Sélectionnez l'item Modifier la lettre de lecteur et les chemins d'accès....

  3. Appuyez sur le bouton Supprimer....

  4. Appuyez sur le bouton Oui dans la fenêtre du message d'avertissement.

  5. Dans l'utilitaire Gestion du disque, la partition n'a plus de lettre.
