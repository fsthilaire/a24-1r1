Nommer une partition

Il est possible de nommer une partition ou de modifier le nom après sa création.

  1. Effectuez un clic-droit sur la partition à nommer.

  2. Sélectionnez l'item Propriétés.

  3. Dans la zone de texte, inscrivez le nouveau nom. Appuyez sur le bouton OK.

  4. Dans l'utilitaire Gestion du disque, la partition affiche son nouveau nom.
