Supprimer une partition

Lorsqu'une partition est supprimée, tous les fichiers de celle-ci ne sont plus disponibles.

Cette action n'est pas réversible.

  1. Effectuez un clic-droit sur la partition à supprimer.

  2. Sélectionnez l'item Supprimer le volume....

  3. Appuyez sur le bouton Oui pour confirmer la suppression.

  4. La partition n'est plus visible dans l'utilitaire Gestion du disque.
