Obtenir la version et l'édition de Windows

1. Paramètres de Windows​

Pour connaitre la version de Windows d'un ordinateur, il faut aller dans Paramètres -> Système -> À propos.

L'information se retrouve sous la section Spécifications de Windows.

L'image ci-dessous provient de Windows 10.

2. Invite de commandes - Version uniquement​

Il est possible d'utiliser la commande ver dans l'invite de commandes pour obtenir le numéro de build.

Le build correspond aux 2 dernières séries de chiffres

Attention de ne pas confondre le 10.0 pour différencier l'édition entre Windows 10 et Windows 11. Les 2 éditions utilisent le même noyau, soit NT 10.0.

Pour connaitre la version de Windows à partir du numéro de build, il suffit de consulter ces sites.

  • Windows 10 : https://learn.microsoft.com/fr-ca/windows/release-health/release-information
  • Windows 11 : https://learn.microsoft.com/fr-fr/windows/release-health/windows11-release-information

3. Invite de commandes - Édition et version​

La commande systeminfo permet d'afficher un rapport de la configuration de l'ordinateur.

Il est possible d'afficher uniquement une section par l'ajout d'un filtre. Le texte varie selon la langue, il faut donc appliquer un filtre différent.

Pour un ordinateur en français.

systeminfo | findstr /B /C:"Nom du système" /B /C:"Version du système"

Pour un ordinateur en anglais.

systeminfo | findstr /B /C:"OS Name" /B /C:"OS Version"
