Autre ports externes

1. Ethernet RJ45​

Le port Ethernet utilise un connecteur RJ45. Ce port permet de relier l'ordinateur à un réseau informatique.

Ce port est similaire au RJ11, mais utilise 8 brins.

La carte réseau peut supporter plusieurs vitesses de transmission.

La norme de nos jours est de 1 Gpbs, mais le standard a longtemps été 100 Mbps.

2. Téléphone RJ11​

Le port téléphonique RJ11 est utilisé par le modem téléphonique. Il permet de relier un ordinateur à une ligne téléphonique.

Le transfert de données utilise les fréquences de la voix, donc le débit est très limité.

Ce port est similaire au RJ11, mais utilise 4 brins.

3. Série​

Le port série est un port de communication très ancien. Il est encore utilisé, mais il est de plus en plus remplacé par le USB.

Il permet de communiquer en série avec un autre appareil.

Il est souvent utilisé pour se connecter directement dans les équipements réseau ou certains contrôleurs.

Le port de communication n'est pas très rapide. La vitesse de transmission se mesurait en baud.

Il ne faut pas le confondre avec le port **VGA **avec le port série DB9. Ils ont une forme similaire, mais ne sont aucunement compatibles.

Le modem téléphonique externe et la souris utilisaient le port série anciennement.

astuce

Pour le différencier rapidement, le port VGA est généralement bleu. C'est un port femelle.

Le port série a seulement 9 broches et c'est un port mâle.

4. Parallèle​

Le port parallèle n'est plus réellement utilisé de nos jours.

Il permettait de communiquer en parallèle avec un autre appareil. Les imprimantes utilisaient ce port anciennement.

5. PS/2​

Le port PS/2 permet de connecter un clavier (mauve) et une souris (vert).

Les ports sont identiques, mais ne sont pas interchangeables. Il est important de respecter la couleur pour le bon périphérique.

Ces ports existent encore sur certaines cartes-mères, mais ils sont rarement utilisés.

6. Audio 3.5mm​

Le port audio 3.5mm permet de connecter un appareil audio analogique.

Le port audio peut être une entrée, une sortie ou bi directionnel (3 canaux).

  • Entrée : Micro
  • Sortie : Haut-Parleur
  • Bi-directionnel : Casque d'écoute
