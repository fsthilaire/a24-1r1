Connecteurs internes

Il existe plusieurs connecteurs internes pour relier différents périphériques.

Chaque connecteur est relié à un BUS sur la carte-mère.

1. IDE​

Le port IDE ou PATA est un câble de transmission de données pour les unités de stockage. Ce port transférait l'information en parallèle.

Depuis plusieurs années, ce port n'est plus utilisé.

Ce port transmet uniquement des données.

Source de l'image : https://commons.wikimedia.org/wiki/File:Ata_20070127_002.jpg

2. SATA​

Le port SATA est un port de communication pour les périphériques de stockage. Il est utilisé pour les unités de stockage (HDD ou SSD) et les lecteurs optiques. Il utilise un mode de transmission en série.

Ce port transmet uniquement des données.

Voici un connecteur SATA un câble.

Source de l'image : https://commons.wikimedia.org/wiki/File:SATA_Data_Cable.jpg

Dans l'image ci-dessous, il s'agit des connecteurs #14.

Image de la carte-mère Optiplex 7010. Source : Dell OptiPlex 9010/7010 Mini-tour Manuel du propriétaire, page 35

3. PCI​

Le port PCI est un port d'ancienne génération. Il existe toujours pour permettre l'utilisation des cartes plus anciennes.

Ce port fournit une alimentation électrique en plus de transmettre des données.

Dans l'image ci-dessous, il s'agit du connecteur #2.

Image de la carte-mère Optiplex 7010. Source : Dell OptiPlex 9010/7010 Mini-tour Manuel du propriétaire, page 35

4. PCI Express​

Il existe plusieurs versions des ports PCI-Express. Ils sont désignés par x1, x4, x8 et x16.

Ce port fournit une alimentation électrique en plus de transmettre des données.

Plus le nombre est élevé, plus le connecteur est long et rapide. Il est important de s'assurer que le x16 ne représente pas uniquement la forme et également la vitesse.

Sur la carte-mère du Optiplex 7010, il y a un port PCI-Express x16, mais son câblage est pour du x4. Sa vitesse sera celle d'un port x4. Il s'agit du port #1 dans l'image ci-dessous.

Les cartes graphiques doivent utiliser le port x16 pour avoir la plus grande vitesse possible. Il s'agit du port #5 dans l'image ci-dessous.

Le port #3 consiste à un PCI-Express x1.

Image de la carte-mère Optiplex 7010. Source : Dell OptiPlex 9010/7010 Mini-tour Manuel du propriétaire, page 35

5. M.2​

Ce port est un port PCI-Express compacte qui permet de connecter des petits périphériques.

Ce port fournit une alimentation électrique en plus de transmettre des données.

Il est utilisé pour les cartes Wi-FI ou les disques durs SSD NVMe.

Ce port n'est pas toujours présent sur la carte-mère, mais il est de plus en plus fréquent. Il est possible d'utiliser un adaptateur qui se connecte dans un port PCI-Express classique pour utiliser un disque dur SSD MVMe.
