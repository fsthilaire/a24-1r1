Manuel technique

Le manuel technique est un document essentiel pour l'informaticien.

La majorité des fournisseurs de pièces et d'ordinateur fournissent un manuel technique qui explique comment effectuer la configuration et l'installation des composantes.

1. Dell Support​

Au Cégep, les ordinateurs sont du fabricant Dell. La procédure démontrée dans ce cours sera pour les ordinateurs Dell, mais la plupart des fournisseurs d'ordinateur ont des outils en ligne pour trouver facilement les manuels techniques.

Les manuels techniques se retrouvent sur le site de Dell Support : https://www.dell.com/support

Il est possible de modifier la langue pour avoir le site dans la langue de notre choix.

  • Français : https://www.dell.com/support/home/fr-ca?lwp=rt
  • Anglais L https://www.dell.com/support/home/en-ca?lwp=rt

2. Service Tag​

Les ordinateurs assemblés par des fournisseurs ont généralement un numéro de série ou d'identification unique pour les identifier.

Pour un ordinateur Dell, il s'agit du Service Tag, ST ou numéro de demande de service en français. Ce code alphanumérique se retrouve sur une étiquette collée à l'ordinateur.

3. Fonctionnement​

Allez sur le site de Dell Support.

Dans la section Identifiez votre produit ou recherchez de l'assistance, inscrivez le Service Tag de l'ordinateur et appuyez sur le bouton Rechercher.

Le site affichera la page du produit. Sélectionnez l'onglet Documentation.

Sur la page documentation, sélectionnez l'onglet Manuels et Documents à gauche.

Ensuite, il faut prendre le document en fonction du format de la tour. Pour cet exemple, le format du boitier est Mini-tour. Cliquez sur le lien Afficher PDF pour ouvrir le document.

Il est possible de choisir la langue du document en utilisant la liste déroulante en haut à gauche.

Conseil

Malheureusement, les manuels en français ne sont pas toujours traduits correctement ou les termes utilisés sont plus européens que canadiens. Il peut être difficile de comprendre leur signification. Beaucoup d'informaticiens préfèrent les manuels techniques en anglais pour faciliter la compréhension des termes.

Par exemple, dans le document français du modèle 7010, les termes Dépose et Pose sont utilisés pour désigner Remove (Enlever) et Install (Installer).
