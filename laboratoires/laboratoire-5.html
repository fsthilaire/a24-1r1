Laboratoire de la semaine 5

Important

Vous devez avoir absolument votre clé USB pour ce laboratoire.

1. Présentation des outils​

L'enseignant vous présentera les outils qui seront utilisés lors de l'assemblage de l'ordinateur.

2. Manuel technique​

Important

Conservez ces documents dans votre ordinateur portable. Il vous sera utile pour le reste de la session.

Identifiez le Service Tag des 3 ordinateurs que vous avez sur votre table de travail.

À partir du site de Dell Support et utilisez le Service Tag pour télécharger le manuel technique (Manuel du propriétaire ou Owner manual) en format PDF pour ces ordinateurs.

Il est possible que vous ayez déjà les documents du modèle Optiplex 9020, car ils étaient dans les exercices du laboratoire 3.

L'explication pour trouver le manuel sur le site de Dell se retrouve dans le site du cours Théorie -> Assemblage et composantes -> Manuel Technique

3. Réparation de l'ordinateur Optiplex 9020​

Pour être en mesure de créer votre Clé USB, vous devez réparer l'ordinateur Dell Optiplex 9020 en installant les pièces manquantes.

Important

Si vous êtes en équipe, il est important de partager les tâches.

Vous êtes libre de partager les tâches à votre guise, mais vous pouvez utiliser cette suggestion.

  • À la section 3, l'étudiant #1 retire les pièces des ordinateurs 7010 et 3620. L'étudiant #2 installe les pièces dans l'ordinateur 9020.

  • À la section 7, l'étudiant #2 retire les pièces de l'ordinateur 9020. L'étudiant #1 installe les pièces dans l'ordinateur 7010 et 3620.

3.1. Bloc d'alimentation - Precision Tower 3620​

Retirez le bloc d'alimentation de l'ordinateur Dell Precision Tower 3620. Son bloc d'alimentation est compatible avec l'ordinateur Optiplex 9020.

Remarquez que le connecteur principal du bloc d'alimentation ne respecte pas le standard P24. Il s'agit d'un format propriétaire à Dell. Il sera nommé Principal P8, car il y a 8 broches.

Installez ce bloc d'alimentation dans l'ordinateur Optiplex 9020.

N'oubliez pas d'utiliser le guide à câble pour le sécuriser lors de l'installation.

Avertissement

Il faut déconnecter tous les 2 connecteurs qui relient le bloc d'alimentation à la carte-mère avant de dévisser.

Il faut dévisser seulement les 4 vis extérieures du bloc d'alimentation.

Les vis intérieures sont pour le ventilateur. Il ne faut pas les dévisser.

astuce

Trouvez la section dans le manuel technique qui explique comment faire cette procédure pour vous aider.

Il y a un verrou qui maintient le bloc d'alimentation en place. Il s'agit du bouton pressoir bleu près du bloc d'alimentation.

3.2. Disque SSD - Dell Precision Tower 3620.​

Le disque qui contient Windows se retrouve dans le SSD.

Le disque SSD est son boitier convertisseur. Le boitier convertisseur est dans son support (bleu). Il n'est pas nécessaire de retirer le support, car il est compatible pour avec le modèle Optiplex 9020.

Prenez également le câble de données SATA qui était utilisé pour ce disque.

Insérez le disque dans l'emplacement à disque du bas dans l'ordinateur Optiplex 9020.

Branchez le câble de données dans le port SATA 0 de la carte-mère. Il s'agit du bleu. N'oubliez pas de brancher le câble alimentation SATA également au disque SSD.

3.3. Mémoire vive - Dell Optiplex 7010​

Il faut installer de la mémoire vive dans l'ordinateur Optiplex 9020. Cet ordinateur utilise de la DDR3.

Le modèle Optiplex 7010 utilise également de la DDR3. Il faut utiliser la mémoire vive de cet ordinateur.

Vous devez l'installer en respectant les règles du double canal. La taille, le fournisseur et le modèle doivent être identiques dans les emplacements DIMM1-DIMM2 et DIMM3-DIMM4.

Avertissement

La mémoire vive de l'ordinateur Precision Tower 3620 utilise de la DDR4. Ce n'est donc pas compatible.

astuce

Pour l'installation, il faut entendre le clic lors de l'insertion.

Appuyez aux 2 extrémités avec vos pouces pour bien l'enfoncer.

Les leviers se relèveront par eux-mêmes. Si un levier n'est pas levé correctement, la mémoire est mal insérée.

3.4. Disque HDD - Dell Optiplex 7010​

Le disque qui contient Ventoy et le fichier ISO de Windows PE se retrouve dans le HDD.

Le HDD est dans son support (bleu). Il n'est pas nécessaire de retirer le support, car il est compatible pour avec le modèle Optiplex 9020.

Prenez également le câble de données SATA qui était utilisé pour ce disque.

Insérez le disque dans l'emplacement à disque du haut dans l'ordinateur Optiplex 9020.

Branchez le câble de données dans le port SATA 1 de la carte-mère. Il s'agit du noir. N'oubliez pas de brancher le câble alimentation SATA également au disque HDD.

3.5. Test - Dell Optiplex 9020​

Refermez le capot de l'ordinateur Optiplex 9020.

Démarrez l'ordinateur. Si vous voyez Windows, vous avez réussi.

Dans le cas contraire, voici des pistes de solution en fonction du message.

  • Si vous voyez le message "No Bootable Device".

    Le Disque SSD n'est pas correctement connecté.

    Vérifiez que le disque SSD a son câble de données et d'alimentation.

    Le câble de données doit être connecté à la carte-mère.

    Le disque étant dans un boitier convertisseur 2.5" à 3.5", il est possible qu'il soit mal inséré à l'intérieur.

  • Si l'ordinateur ne démarre pas.

    Le bloc d'alimentation n'est pas connecté correctement.

    Vérifiez si le connecteur principal P8 et le connecteur du processeur P4 sont insérés correctement.

  • Si l'ordinateur émet un bip sonore.

    La mémoire vive est mal insérée.

    Il faut la retirer et recommencer.

4. Création de la clé USB démarrable​

Vous devez faire cette étape pour la clé USB de chacun des étudiants de l'équipe.

Important

Assurez-vous de brancher votre clé USB dans un port USB 3.0.

Si vous ne voyez pas le lecteur D: ou un lecteur nommé 1R1, le disque HDD n'est pas connecté correctement.

4.1. Ventoy​

Le logiciel Ventoy permet de créer une clé USB démarrable.

Le logiciel se retrouve dans le dossier D:\outils\ventoy\.

Procédure

Référez-vous à la section 3 du document Procédures -> Utilitaires -> Ventoy pour savoir comment créer la clé.

4.2. Copie du ISO​

Vous devez copier le fichier ISO de Windows PE sur la partition Ventoy de votre clé USB.

Voici l'emplacement du fichier ISO : D:\iso\WinPEx64_A24.iso

Procédure

Référez-vous à la section 4 du document Procédures -> Utilitaires -> Ventoy pour voir l'explication.

4.3. Test des clés​

Branchez votre clé USB dans un port USB 3.0.

Sélectionnez le fichier WinPE_x64_A24.iso.

Le système chargera en mémoire une version créée pour ce cours de Windows PE.

Procédure

Référez-vous à la section 5 du document Procédures -> Utilitaires -> Ventoy pour voir l'explication.

5. Analyse du matériel​

Utilisez les HWiNFO et RamCheck pour faire l'analyse du matériel à partir de Windows PE.

Notez et conservez les réponses ci-dessous. Vous pouvez prendre un photo de l'écran qui affiche l'information.

Vous devrez les utiliser lors du prochain laboratoire.

Procédure

Référez-vous aux documents ci-dessous :

  • Référez-vous à la section 4 du document Procédures -> Détection du matériel -> HWiNFO pour avoir les explications sur le fonctionnement du logiciel.

    • La commande à inscrire pour exécuter le logiciel est hwinfo64

  • Procédures -> Détection du matériel -> WMIC (RamCheck) pour avoir les explications sur le fonctionnement du logiciel.

    • La commande à inscrire pour exécuter le logiciel est ramcheck

N'hésitez pas à demander l'aide de l'enseignant pour vous guider.

  1. Identifiez la configuration de la mémoire vive.
    1. Identifiez la capacité totale installée dans l'ordinateur.
    2. Identifiez la capacité, le fournisseur et la marque de chaque barrette.
    3. Est-ce qu'elle respecte les principes du double canal ? Corrigez le problème si ce n'est pas correct.
  2. Identifiez l'information du disque SSD.
    1. Modèle
    2. Capacité
    3. Technologie
  3. Identifiez l'information du disque HDD.
    1. Modèle
    2. Capacité
    3. Technologie
  4. Identifiez l'information du processeur.

    1. Nom du processeur

    2. Fréquence

    3. Nombre de coeurs

    4. Nombre de processeurs logiques (Thread)

  5. Identifiez le nombre de cartes réseau.

6. Réinstallation​

Afin de se pratiquer davantage, il faut remettre le matériel utilisé dans le bon ordinateur.

6.1. Optiplex 7010​

  • Mémoire vive
  • Disque HDD
    • Dans l'emplacement du haut.
    • Câble SATA noir connecté au HDD et sur la carte-mère dans le port SATA 1 (noir).

6.2. Precision Tower 3620​

  • Bloc d'alimentation
    • N'oubliez pas les 4 vis.
    • Branchez le connecteur principal P8.
    • Branchez le connecteur pour l'alimentation du processeur P4.
    • Utilisez le guide à câble pour le connecteur principal P8.
  • Disque SSD
    • Dans l'emplacement du bas.
    • Câble SATA (bleu) connecté au HDD et sur la carte-mère dans le port SATA 0 (bleu).

7. Extra​

Demandez à l'enseignant les pièces nécessaires pour d'autres configurations.
