Laboratoire de la semaine 8

Vous devez avoir absolument votre clé USB pour ce laboratoire.

Il est important que les 2 étudiants de l'équipe effectuent des manipulations.

Seulement l'étudiant qui a le bracelet antistatique est autorisé à installer/retirer la pièce de l'ordinateur.

Voici l'emplacement des pièces dans votre espace de travail

  • 1er tiroir
    • Clavier
    • Souris
    • Tournevis
  • 2e tiroir
    • Mémoire vive
    • Carte PCI/PCIe (3)
    • Carte graphique (1)
    • Boite
      • Câble de données SATA (7)
      • Protecteur PCI (8)
      • Protecteur du cadre avant (4)
      • Vis (8)
  • 3e tiroir
    • Disque SSD (4)
    • Disque HDD (5)
    • Boitier convertisseur (2)
    • Support bleu pour disque interne (4)
  • Tablette
    • Bloc d'alimentation (2)
    • Lecteur DVD
    • Tiroir SATA
    • Disque Seagate 80 Go HDD Format 3.5"
Important

Faites l'inventaire des pièces nécessaires avant de débuter une configuration.

Rangez les pièces non nécessaires dans le bon emplacement.

Si vous avez beaucoup de pièces non nécessaires sur la table, vous risquez de les abimer et de les chercher inefficacement.

Si vous manquez d'espace sur votre table de travail, mettez les pièces à utiliser sur la tablette.

Rangez vos sacs sous la table de travail.

Utilisez un portable par équipe pour libérer de l'espace sur la table de travail.

1. Obtenir les configurations mystères​

Voici un petit défi pour obtenir la configuration mystère.

Vous devez télécharger le fichier configurations_l8-94ed9e3a6a3c4277c9b92575b76fa8ac.zip. Ce fichier contient le document configurations_l8.pdf

Le fichier Zip est protégé par un mot de passe. Pour obtenir le mot de passe, vous devez répondre aux questions ci-dessous en utilisant l'ordinateur Dell Precision Tower 3420, format Compact.

Pour répondre à ces questions, vous devez télécharger le manuel technique, utilisez les 3 outils diagnostiques et l'invite de commandes. Il est interdit d'ouvrir le boitier pour obtenir les réponses.

La réponse de chacune des questions permettra de construire le mot de passe.

Avant de débuter, téléchargez sur le site de Dell Support le manuel technique en français pour cet ordinateur.

Vous avez 50 minutes pour trouver le code. Après ce délai, l'enseignant vous donnera le mot de passe du document.

Le code est composé uniquement de chiffre. Il a une longueur de 20 chiffres.

astuce

Vous avez droit à 2 jokers par équipe.

Le premier permet d'avoir la réponse à une question.

Le deuxième permet d'avoir un indice à une question.

Utilisez-les au bon moment.

Également, il est possible de sauter une question et d'y revenir.

1.1. Manuel technique​

Pour les questions ci-dessous, vous devez utiliser les pages 24 et 25 du manuel technique.

Question 1​

Combien y a-t-il d'emplacements DIMM pour cet ordinateur ?

Question 2​

Est-ce qu'il y a un emplacement PCIe x1 dans cet ordinateur ?

Non = 0

Oui = 1

Question 3​

Est-ce qui y a un emplacement PCI pour cet ordinateur ?

Non = 0

Oui = 1

Question 4​

Quelle est la couleur du port SATA 1 ?

Noir = 1

Blanc = 2

Bleu = 3

Question 5​

Quel est le numéro de la charte qui représente l'emplacement pour la carte graphique ?

Question 6​

Quel est le numéro de la charte qui représente le connecteur d'alimentation principal P8 de la carte-mère ?

1.2. Outils diagnostiques​

Vous devez utiliser les outils diagnostiques pour répondre aux questions ci-dessous.

Question 7​

Quelle est la touche pour accéder au BIOS ?

F1 = 1

F2 = 2

F3 = 3

...

F12 = 12

Question 8​

Quelle est la marque de l'unité de stockage connecté dans l'ordinateur ?

Western Digial = 1

Samsung = 2

Seagate = 3

Sandisk = 4

Micron = 5

Intel = 6

Question 9​

Quelle est la capacité de l'unité de stockage ?

Vous devez inscrire uniquement les chiffres. Par exemple, si la capacité est de 128 Go, il faut inscrire 128.

Question 10​

L'unité de stockage est connecté dans quel port SATA ?

SATA 0 = 0

SATA 1 = 1

SATA 2 = 2

SATA 4 = 4

Question 11​

Est-ce que le lecteur DVD est correctement installé ?

Non = 0

Oui = 1

Question 12​

Est-ce que la mémoire vive est correctement installée ? Cette carte-mère utilise le système de numérotation de DIMM que nous avons vu en classe.

Non = 0

Oui = 1

Question 13​

Quelle génération de mémoire vive qui est installée dans l'ordinateur ?

DDR3 = 3

DDR4 = 4

Question 14​

Combien y a-t-il de barrettes de mémoire dans l'ordinateur ?

1.3. Invite de commandes - WinPE​

Vous devez utiliser l'invite de commandes dans Windows PE pour répondre aux questions.

Quel est le nom du sous-dossier qui se retrouve dans le dossier x:\Users\.

Question 16​

Quelle est la taille du fichier x:\Windows\Boot\Boot\BootDebuggerFiles.ini ?
