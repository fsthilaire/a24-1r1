Laboratoire de la semaine 11

Dans ce laboratoire, vous allez restaurer votre installation de Windows qui a été créé sur un disque HDD sur un disque SSD.

Ensuite, vous configurez des périphériques pour que Windows soit en mesure de les utiliser.

Le disque Seagate 500 Go HDD 3.5" contient Windows 11. Vous devez faire une image du Seagate 500 Go HDD 3.5". Ensuite, vous devez restaurez cette image sur le Samsung 250 Go SSD 2.5".

Astuce

Vous pouvez utiliser la même technique avec votre propre ordinateur, si vous voulez faire une mise à niveau d'un HDD vers un SSD sans tout réinstaller. Il faut que la somme de la taille de toutes les partitions du disque HDD soit égale ou inférieure à la taille totale du disque SSD.

Le disque Seagate 500 Go HDD 3.5" utilise 100 Go pour toutes ses partitions. C'est pour cette raison qu'il est possible de le restaurer sur le Samsung 250 Go SSD 2.5".

Le disque Seagate 500 Go HDD 3.5" utilise le port SATA 0.

Le disque Samsung 250 Go SDD 2.5" utilise le port SATA 1.

1. Vérifier Windows​

Démarrez l'ordinateur pour vérifier que Windows est bien installé sur le disque Seagate 500 Go HDD 3.5".

Pourquoi Windows ne démarre pas ?

La réponse est dans le BIOS.

Indice

Pensez à ce que vous faisiez pour vérifier si une unité de stockage était bien connectée.

2. Création de l'image​

Insérez le disque Micron 256 Go SSD 2.5" qui a le numéro IMPAIR dans le tiroir SATA du haut.

Afin de limiter le nombre de disques visibles dans Paragon, désactivez le port SATA 1 dans le BIOS pour ne pas voir le disque Samsung 250 Go SDD 2.5".

Démarrez l'ordinateur avec Paragon à partir de votre clé USB.

Vous devez créer l'image sur la partition qui correspond à votre groupe. Créez un dossier avec le nom de votre table. Si votre table a le numéro 9, vous devez créer le dossier Table9.

Éteignez l'ordinateur lorsque la création est terminée.

3. Restauration de votre image​

Allez dans le BIOS, et désactivez le port SATA 0 et activez le port SATA 1.

Vous devez restaurez votre image Paragon sur le disque Samsung 250 Go SSD 2.5" qui se retrouve dans le boitier de l'ordinateur. Il est dans le port SATA 1.

Démarrez l'ordinateur avec Paragon à partir de votre clé USB.

Restaurez votre image sur le disque Samsung 250 Go SSD 2.5".

Éteignez l'ordinateur lorsque la restauration est terminée.

Retirez le disque Micron 256 Go SSD 2.5" qui a le numéro IMPAIR dans le tiroir SATA du haut.

Retirez votre clé USB.

Démarrez l'ordinateur et ce sera le Windows du disque SSD qui sera utilisé.

4. Configuration​

4.1. Modifier le nom de l'ordinateur​

Vous devez modifier le nom de l'ordinateur pour Gr#Tb@.

Le # représente votre numéro de groupe et le @ représente votre table de travail qui est inscrite sur la tablette.

Exemple, si vous êtes le groupe 5 et vous avez la table 2, l'ordinateur doit se nommer Gr5Tb2

Il est important de respecter les majuscules et minuscules.

Rappel

Le nom de l'ordinateur est le nom qui sera visible sur le réseau. Il représente l'installation de Windows.

Le nom de l'utilisateur est le nom pour se connecter dans Windows.

La procédure se retrouve dans la section Procédures -> Windows -> Renommer un ordinateur.

4.2. Disposition du clavier​

Ajoutez la disposition Anglais (États-Unis).

Il faut désactiver les raccourcis CTRL+MAJ et ALT Gauche + MAJ pour le changement de disposition.

4.3. Explorateur de fichiers​

Il faut afficher les extensions des fichiers.

Il faut afficher les fichiers masqués.

Il faut mettre Ce PC et non Accès rapide à l'ouverture de l'Explorateur de fichier.

4.4. Gestion de l'alimentation​

Il faut éteindre les écrans après 30 minutes.

Il faut mettre l'ordinateur en veille après 3 heures.

5. Installation de périphériques​

Note

Référez-vous aux procédures qui se retrouvent dans la section Procédures -> Windows -> Gestionnaire de périphérique.

Insérez le disque Micron 256 Go SSD 2.5" qui a le numéro PAIR dans le tiroir du haut.

Vous devez installer différents périphériques.

Certaines pièces doivent être partagées entre les équipes.

Créez le dossier c:\pilotes.

Copiez tout le contenu du dossier drv qui se retrouve dans la partition Drivers du disque Micron 256 Go SSD 2.5" PAIR

Clé USB

Pendant cette section, utilisez votre ordinateur portable pour copier le fichier W11_1R1.iso sur la 2e clé USB si ce n'est pas déjà fait.

5.1. Carte réseau PCI #1 - StarTech​

Regardez la pièce. Il y a un identifiant avec le site web du fournisseur et un numéro de pièce.

Avertissement

Recherchez le pilote sur les sites web du fournisseur.

Vous devez extraire le fichier zip dans le dossier c:\pilotes.

N'utilisez pas le programme d'installation Setup.exe.

Trouvez le dossier qui contient le fichier inf pour la version la plus récente de Windows et pour la bonne architecture.

La pièce doit être correctement installée.

5.2. Carte sans-fil USB​

Récupérez l'identifiant de la pièce dans le gestionnaire de périphérique.

Le pilote se retrouve dans le sous-dossier de c:\pilotes qui a le même identifiant.

Installez le pilote.

La pièce doit être correctement installée.

5.3. Carte HDTV PCI-Express​

Demandez la pièce à un collègue si vous ne l'avez pas.

Avertissement

Recherchez le pilote sur les sites web du fournisseur.

Le pilote sera un fichier exe. Le pilote s'installera en mode assisté, comme pour la carte réseau Dell de la semaine dernière.

Il suffit d'exécuter le programme pour installer le pilote.

Installez le pilote.

La pièce doit être correctement installée.

5.4. Webcam​

Demandez la pièce à un collègue si vous ne l'avez pas.

Récupérez l'identifiant de la pièce dans le gestionnaire de périphérique.

Le pilote se retrouve dans le sous-dossier de c:\pilotes qui a le même identifiant.

Installez le pilote.

La pièce doit être correctement installée.

5.5. Carte réseau PCI #2 - Linksys​

Demandez la pièce à un collègue si vous ne l'avez pas.

Récupérez l'identifiant de la pièce dans le gestionnaire de périphérique.

Le pilote se retrouve dans le sous-dossier de c:\pilotes qui a le même identifiant.

Installez le pilote.

La pièce doit être correctement installée

6. Fichier ISO Ubuntu​

Vous devez copier sur les 2 clés USB le fichier ISO pour Ubuntu. Il sera nécessaire pour le prochain cours théorique de la semaine 11.

Copiez le fichier ubuntu-22.04.3-desktop-amd64.iso qui se retrouve sur la partition ISO du disque Micron 256 Go SSD 2.5" PAIR vers votre clé USB (Partition Ventoy).

7. Réinstallez Windows​

Important

Assurez-vous de retirer le disque du tiroir SATA.

Assurez-vous que le port SATA 0 est toujours désactivé dans le BIOS.

Si le temps vous le permet, réinstallez Windows sur le disque Samsung 250 G0 SSD 2.5". L'opération sera beaucoup plus rapide, car vous allez installer Windows sur le disque SSD.

Lors de la création des partitions, les partitions de l'installation précédente seront présentes. Il est important de tous les effacer.

Windows doit être installé sur une partition de 60 Go. Effectuez la conversion en Mo correctement en utilisant le facteur 1024.

Le nom d'utilisateur doit être Admin1321 et il ne doit pas avoir de mot de passe.

La disposition du clavier doit être Français (Canada).

Vous devez installer les pilotes de la carte réseau pour être en mesure d'utiliser Dell Support Assist.

8. Réinitialisation du disque Samsung 250 Go SSD 2.5"​

Important

Assurez-vous de retirer le disque du tiroir SATA.

Assurez-vous que le port SATA 0 est toujours désactivé dans le BIOS.

Démarrez avec Windows PE à partir de votre clé USB.

Lorsque vous êtes dans Windows PE, retirez votre clé **USB **pour éviter de la réinitialiser par erreur.

Il faut réinitialiser le disque Samsung 250 Go SSD 2.5" à partir de DiskPart.

Tapez la commande diskpart pour démarrer le logiciel.

La commande ci-dessous permet d'afficher les unités de stockage.

list disk

Identifiez le numéro correspondant au disque de 250 Go.

La commande ci-dessous permet de sélectionner le disque dur. Il faut remplacer le ### par le numéro du disque dur identifié à l'étape précédente.

select disk ###

Exemple

select disk 0

Ensuite, les commandes ci-dessous permettent la réinitialisation.

clean
convert gpt

Redémarrez l'ordinateur. Vous devez avoir la mention No bootable device.
